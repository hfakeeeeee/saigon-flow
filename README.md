# saigon-flow
Guide the rush through a city of alleys, bridges, and rain.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

The Vite base path is already set to `/saigon-flow/` in `vite.config.ts`.

```bash
npm run deploy
```

## Firebase later

The prototype is offline-first for now. Firebase is a good next step for:

- global leaderboards
- cloud saves
- daily challenge seeds and results
- anonymous user profiles
- lightweight gameplay analytics
